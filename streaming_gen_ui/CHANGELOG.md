## 0.0.1-alpha.1

* **Initial Release**: First public release of streaming_gen_ui
* Added `UiStreamer` class for handling real-time UI data streaming
* Added `StreamingWidgetDecoder` for decoding streamed JSON data
* Integrated with `llm_json_stream` package for LLM output support
* Support for streaming JSON to Flutter widgets conversion
* Full Dart and Flutter SDK support
* Comprehensive documentation and examples
* Multi-platform compatibility (iOS, Android, Web, macOS, Linux, Windows)
